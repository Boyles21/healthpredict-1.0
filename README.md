# HealthPredict

An AI-powered web application for the early risk assessment of **Polycystic Ovary Syndrome (PCOS)** and **Uterine Fibroids** using Machine Learning. HealthPredict provides fast, user-friendly health assessments, personalized recommendations, and an intuitive dashboard designed to support early awareness and encourage timely medical consultation.

> **Disclaimer:** HealthPredict is a screening support tool and is **not** intended to replace professional medical diagnosis or treatment.

---

## Features

* AI-powered **PCOS Risk Prediction**
* AI-powered **Uterine Fibroid Risk Prediction**
* Separate assessment workflows for each condition
* Personalized health recommendations
* Interactive dashboard
* Secure Firebase Authentication
* Responsive design for desktop, tablet, and mobile
* Prediction history management
* Modern animated user interface

---

## Machine Learning Models

### PCOS Prediction Model

* Algorithm: Random Forest Classifier
* Dataset Size: 541 patient records
* Features: 12
* Separate preprocessing pipeline
* Independent prediction endpoint

### Fibroid Prediction Model

* Algorithm: Random Forest Classifier
* Dataset Size: 1,538 patient records
* Features: 15
* Independent preprocessing pipeline
* Independent prediction endpoint

Both models operate independently and do not interfere with one another.

## Installation

Clone the repository

```bash
git clone [https://github.com/Boyles21/healthpredict-1.0].git
```

Navigate into the project

```bash
cd healthpredict
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Start the backend

```bash
npm run server
```

---

## Model Persistence

To improve startup performance, both Random Forest models are serialized after training.

On startup the application:

* Loads existing JSON models if available.
* Falls back to training from the CSV datasets if models do not exist.
* Automatically saves newly trained models for future use.

This significantly reduces deployment startup time while preserving prediction accuracy.

## User Workflow

1. Sign in securely using Google Authentication.
2. Choose either:

   * PCOS Assessment
   * Fibroid Assessment
3. Complete the health questionnaire.
4. Receive an AI-generated risk assessment.
5. Review personalized recommendations.
6. Save and revisit previous assessments.
