# HealthPredict

AI-Powered PCOS Risk Assessment and Health Monitoring System

## Overview

HealthPredict is a web-based healthcare application that leverages Machine Learning techniques to assess the risk of Polycystic Ovary Syndrome (PCOS) based on clinical symptoms, lifestyle factors, and menstrual health indicators.

The system utilizes a Random Forest Classifier model to analyze user-provided health information and generate risk predictions with supporting recommendations.

## Key Features

* Secure user authentication using Firebase Authentication
* Personal health profile management
* AI-powered PCOS risk prediction
* Prediction confidence scoring
* Historical assessment tracking
* Risk trend visualization
* Cloud-based data storage using Firebase Firestore
* Excel dataset upload and analysis
* Responsive user interface for desktop and mobile devices

## System Architecture

1. User enters health indicators and symptoms.
2. Frontend sends structured data to the prediction API.
3. Backend preprocesses inputs and performs model inference.
4. Random Forest model generates risk prediction.
5. Results and recommendations are returned to the user.
6. Assessment history is securely stored in Firebase Firestore.

## Installation

### Prerequisites

* Node.js
* npm

### Clone Repository

```bash
git clone https://github.com/yourusername/healthpredict.git
cd healthpredict
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file and add:

```env
GEMINI_API_KEY=your_api_key
```

### Start Development Server

```bash
npm run dev
```
